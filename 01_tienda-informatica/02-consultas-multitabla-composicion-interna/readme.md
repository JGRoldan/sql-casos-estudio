# Consultas multitabla – Composición interna

Listado de consultas multitabla utilizando composición interna (INNER JOIN),
aplicadas sobre la base de datos de tienda de informática.

---

## Consultas

- [X] Devuelve una lista con el nombre del producto, precio y nombre de fabricante de todos los productos de la base de datos.

- [X] Devuelve una lista con el nombre del producto, precio y nombre de fabricante de todos los productos de la base de datos.  
      Ordene el resultado por el nombre del fabricante, por orden alfabético.

- [X] Devuelve una lista con el identificador del producto, nombre del producto, identificador del fabricante y nombre del fabricante, de todos los productos de la base de datos.

- [X] Devuelve el nombre del producto, su precio y el nombre de su fabricante, del producto más barato.

- [X] Devuelve el nombre del producto, su precio y el nombre de su fabricante, del producto más caro.

- [X] Devuelve una lista de todos los productos del fabricante Lenovo.

- [X] Devuelve una lista de todos los productos del fabricante Crucial que tengan un precio mayor que 200€.

- [X] Devuelve un listado con todos los productos de los fabricantes Asus, Hewlett-Packard y Seagate.  
      Sin utilizar el operador IN.

- [X] Devuelve un listado con todos los productos de los fabricantes Asus, Hewlett-Packard y Seagate.  
      Utilizando el operador IN.

- [X] Devuelve un listado con el nombre y el precio de todos los productos de los fabricantes cuyo nombre termine por la vocal e.

- [X] Devuelve un listado con el nombre y el precio de todos los productos cuyo nombre de fabricante contenga el carácter w en su nombre.

- [X] Devuelve un listado con el nombre de producto, precio y nombre de fabricante, de todos los productos que tengan un precio mayor o igual a 180€.  
      Ordene el resultado en primer lugar por el precio (en orden descendente) y en segundo lugar por el nombre (en orden ascendente).

- [X] Devuelve un listado con el identificador y el nombre de fabricante, solamente de aquellos fabricantes que tienen productos asociados en la base de datos.
